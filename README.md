# dp - React Landing Page

A standalone React + TypeScript landing page built with Vite, Tailwind CSS, and Lucide icons. It contains no Next.js dependency or framework-specific routing.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown by Vite.

## Production build

```bash
npm run build
npm run preview
```

The optimized static output is generated in `dist/` and can be deployed to Netlify, Vercel, Cloudflare Pages, GitHub Pages, or any static host.

## Main editable files

- `src/data/site.ts` — brand, product, workflow, benefit, and trust content
- `src/App.tsx` — page sections and editorial layout
- `src/index.css` — complete visual system and responsive styling
- `src/components/product-render.tsx` — CSS-built placeholder product renders
- `src/components/contact-form.tsx` — demonstration request form

All regulatory statements, metrics, distributor logos, testimonials, specifications, and measurements are marked as editable placeholders and must be replaced with verified information before publication.
